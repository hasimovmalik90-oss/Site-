## Воронка 1: Тест «Родовой сценарий»

**Путь пользователя:**
1. Трафик из Instagram → попадает на лид-форму `kseniiamikaelyan.com/rod/`
2. Заполняет форму (имя, email, ситуация) → данные уходят в Formspree (`mojknovz`, источник: Instagram — РОД)
3. Экран «Готово, проверяй почту» → получает ссылку на тест на email
4. Переходит на страницу теста → `kseniiamikaelyan.com/test_rodovoy/`
5. Отвечает на 25 вопросов (все обязательные)
6. Ответы отправляются в Formspree (текст каждого ответа + контакт)
7. Автоматический редирект → `kseniiamikaelyan.com/rasshifrovka/`
8. На расшифровке — тарифы и форма записи

**Что реализовано:**
- Лид-форма `/rod/`: имя, email, ситуация (4 варианта + «Другое»), отправка в Formspree, экран успеха
- Тест на 25 вопросов с обязательным заполнением всех
- Светящиеся бриллианты-разделители между блоками
- Отправка полного текста ответов (не процентов) в Formspree
- Каждый ответ — отдельное поле с текстом
- Контактная форма: имя, Telegram, email
- Редирект на страницу расшифровки после отправки
- Страница расшифровки с тарифами и кнопкой записи в золотом стиле

## Риски, сложности и человеческий фактор

### 1. Коммуникация с заказчиком

| Проблема | Последствия |
|----------|-------------|
| «Мне не нравится, переделайте» | +30–50% времени на правки дизайна |
| «А можно чуть-чуть поменять...» (x15 раз) | Скоуп-крип, бюджет улетает |
| «Я не это имел(а) в виду» | Переделка целых блоков с нуля |
| Заказчик пропадает на 3–7 дней | Проект растягивается с 3 недель до 2 месяцев |
| Правки через голосовые в WhatsApp | Потеря информации, разночтения |

**Реальный множитель: x1.5–2.0 к срокам**

### 2. Дизайн

| Проблема | Последствия |
|----------|-------------|
| «Сделайте красиво» без ТЗ | 3–5 итераций макета вместо 1 |
| Анимации «как на том сайте» | +10–20 часов на реверс-инжиниринг |
| Дизайнер рисует то, что невозможно сверстать | Фронтенд ругается, переделка макета |
| Шрифты/цвета меняются в процессе | Переверстка всех страниц |
| «А на телефоне выглядит не так» | Адаптив — это +40–60% к верстке |

**Реальный множитель: x1.3–1.8 к бюджету дизайна**

### 3. Разработка

| Проблема | Последствия |
|----------|-------------|
| Кроссбраузерность (Safari, Firefox) | +8–15 часов на фиксы |
| Formspree лимиты (50 отправок/мес на бесплатном) | Нужен платный план или альтернатива |
| Canvas-анимации тормозят на слабых устройствах | Оптимизация или отключение на мобилках |
| GitHub Pages не поддерживает серверную логику | Ограничения архитектуры |
| Тесты с 25 вопросами — баги в логике подсчета | +5–10 часов на отладку и QA |
| «Почему форма не отправляется?» | CORS, спам-фильтры, опечатки в endpoint |

**Реальный множитель: x1.2–1.5 к часам разработки**

### 4. Человеческий фактор исполнителей

| Проблема | Последствия |
|----------|-------------|
| Фрилансер пропал / заболел / «семейные обстоятельства» | Проект стоит, ищем замену (+1–2 недели) |
| Фрилансер взял 5 проектов параллельно | Ваш проект в низком приоритете |
| «Я почти закончил» (уже 2 недели) | Классика фриланса |
| Разработчик написал код без комментариев и уволился | Новый разработчик не может разобраться |
| Дизайнер и верстальщик не общаются | Результат не совпадает с макетом |
| Тестирование? «А что тестировать, и так работает» | Баги в проде, стыд перед клиентами |

**Реальный множитель: x1.3–2.0 к срокам**
